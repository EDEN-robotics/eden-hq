## ğŸ¨ Understanding the Format

### TODO.md Format:

```

### ROADMAP.md Format:
```

**Status Options:**
- `:done` = Finished
- `:active` = In progress
- `:crit` = Critical/urgent
- `:milestone` = Important milestone
